Poubelle Intelligente Connectée

Projet de groupe Systèmes Embarqués Février 2026

Système IoT automatisant l’ouverture d’une poubelle via capteur ultrason à 5cm. Affichage d’état sur LCD I2C avec LED témoin et remontée des données température/humidité sur interface web hébergée par l’ESP32.

Mon rôle : Développement du code C++ Arduino, intégration du serveur web embarqué et tests des capteurs ultrason, DHT11 et LCD.

Technologies : C++ Arduino ESP32 HTML/CSS DHT11 LCD I2C

Gestion de Répertoire Téléphonique

Projet de groupe Algorithmique 2026

Application console en langage C avec base de données SQL pour gérer un carnet d’adresses. CRUD complet : création, lecture, modification, suppression et recherche de contacts via menu interactif.

Mon rôle : Conception de la structure CONTACT, implémentation des fonctions Saisie_Contact(), Search_Contact() et gestion des requêtes SQL pour la persistance des données.

Technologies : Langage C SQL Structures Pointeurs

AQUASCAN - Business Plan

Projet de groupe Projet conceptuel Entrepreneuriat 2026

Étude et conception d’un service de distribution d’eau potable via kiosques IoT connectés. Travail réalisé dans le cadre du cours d’Entrepreneuriat : étude de marché, modèle économique et spécifications techniques pour zones à accès limité.

Mon rôle : Participation à l’étude de marché, définition des fonctionnalités techniques du kiosque IoT et élaboration du modèle économique.

Compétences mobilisées : Business Model Étude de marché Cahier des charges IoT

Sécurisation Web avec OpenSSL

Projet individuel Sécurité Avancée Février 2026

Mise en œuvre des mécanismes de cryptographie avec OpenSSL sous Linux. Chiffrement symétrique AES/3DES, asymétrique RSA 2048 bits, hachage SHA256, signature numérique et création de certificats X.509 auto-signés pour sécuriser Apache2 en HTTPS.

Technologies : OpenSSL Linux CLI AES/RSA SHA256 X.509 Apache2

Technologies : Langage C Code::Blocks Algorithmique Analyse Numérique
